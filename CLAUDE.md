# Claudinite

> ℹ️ The owner's personal preferences and the active packs' rules are injected automatically at
> session start — they are already in context above. Honor them; there is nothing to go read.
> The conformance checks run on their own when you finish; each failure message carries its rule
> and its fix.

The project declares its packs — and every per-project setting — in `.claudinite-checks.json`;
the project's own packs live in `.claudinite/local_packs/<name>/`.
